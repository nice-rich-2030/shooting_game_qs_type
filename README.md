# R-TYPE Clone

日本のアーケードゲーム「R-TYPE」に似た横スクロールシューティングゲームです。Pygameで実装されています。

## 特徴

### R-TYPEの主要な要素を実装

- **Forceオーブシステム** - プレイヤー機の前後に装着可能な無敵の球体
  - 前方装着：敵を破壊しながら前進
  - 後方装着：背後からの攻撃を防御
  - 分離：独立して移動し、敵弾を吸収

- **チャージショット** - 3段階のパワーアップ可能な強力な攻撃
  - レベル1（0.5秒）：通常の2倍サイズ
  - レベル2（1.0秒）：3倍サイズ、2体貫通
  - レベル3（1.5秒）：5倍サイズ、無限貫通

- **4種類の敵**
  - ストレート型：直進する基本的な敵
  - ウェーブ型：波型の軌道で移動し、弾を発射
  - 突撃型：プレイヤーに向かって突進
  - タンク型：耐久力が高く、3方向に弾を発射

- **Wave システム** - 時間経過とともに敵の種類と難易度が変化
  - Wave 1（0-30秒）：ストレート型のみ
  - Wave 2（30-60秒）：ウェーブ型が登場
  - Wave 3（60-90秒）：突撃型が登場
  - Wave 4（90秒以降）：全種類混合、難易度上昇

- **パワーアップアイテム**
  - Force（橙色）：Forceオーブを取得
  - Speed（緑色）：移動速度アップ
  - Power（赤色）：攻撃力アップ

## 操作方法

| キー | 機能 |
|------|------|
| ↑↓←→ | プレイヤー移動（8方向） |
| Z | 通常射撃（押し続けると連射） |
| X | チャージショット（長押しでチャージ、離して発射） |
| C | Force装着位置切り替え（前⇔後⇔分離） |
| R | リスタート（ゲームオーバー時） |
| ESC | 終了 |

## ゲームのコツ

1. **Forceを活用する**
   - 前方装着で敵を直接破壊
   - 後方装着で背後の安全確保
   - 分離して敵弾を吸収（Forceは無敵）

2. **チャージショットを使いこなす**
   - 敵が多い時はレベル3で一掃
   - 画面にチャージゲージが表示される
   - 色が白→黄→赤と変化

3. **敵のパターンを覚える**
   - ストレート型：簡単、弾を撃たない
   - ウェーブ型：軌道が予測しやすい
   - 突撃型：早めに撃破
   - タンク型：HP高い、要注意

4. **パワーアップを集める**
   - 敵を倒すと20%の確率でドロップ
   - Forceは最優先で取得

## 実装の詳細

### 技術仕様

- 画面サイズ：800x600ピクセル
- フレームレート：60 FPS
- グラフィック：シンプルな幾何図形
  - プレイヤー：青い三角形
  - Force：オレンジの円（中心に白い核）
  - 敵：タイプごとに異なる形状と色

### ゲームバランス

- プレイヤー初期ライフ：3
- 被弾後の無敵時間：2秒
- 敵出現間隔：約1.5秒（ランダム変動あり）
- パワーアップドロップ率：20%
